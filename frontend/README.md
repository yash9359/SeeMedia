# SeeMedia Frontend

This folder contains the React + Vite client for SeeMedia.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Environment Variables

- `VITE_API_BASE_URL`
- `VITE_BACKEND_URL`

If not provided, the app uses built-in localhost/production fallbacks.
